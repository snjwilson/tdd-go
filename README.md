# Learn Go With Tests

This repository contains code and exercises from the "Learn Go With Tests" tutorial.

## Getting Started

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/learngowithtests.git
    cd learngowithtests
    ```

2. Run tests:
    ```sh
    go test ./...
    ```
